LED Cube
I also built a 4x4x4 led cube. The led cube had a coin acceptor attached to it on the side. The user would pay a quarter, select a led show, and watch the LED show on the cube.

Python to VHDL
There is a function in Simulink that converts a circuit into VHDL. A company has asked my team to see if Python could be used instead. The attempt is to find a way to take Python and sucessfully convert it to VHDL. Using an open source tool, I was able to create a program and GUI that allowed users to take Python code and convert it to VHDL and Verilog.
