Fun Face Cam I worked on a simple BeagleBone project that allows the user to take a picture and add a mustache, hat, or sunglasses to their face. Click here for more details.

LED Cube I also built a 4x4x4 led cube. The led cube had a coin acceptor attached to it on the side. The user would pay a quarter, select a led show, and watch the LED show on the cube.

Python to VHDL Currently, there is a function in Simulink that converts a circuit into VHDL. A company has asked my team to see if Python could be used instead. The attempt is to find a way to take Python and sucessfully convert it to VHDL. I am currently working on this project.
